Hay cientos de repositorios y guías sobre IA ahí fuera. La mayoría son puro ruido.
He borrado el 90% de mis marcadores para traerte solo lo que realmente uso.

Si estás intentando pasar de hacer prompts básicos a construir Agentes (o pelearte con Claude Code), no necesitas tener 50 pestañas abiertas.

Esta es mi lista curada y mi flujo de trabajo real para no perder tiempo:

## 1. El "Paso Cero" (Anthropic Skills)
Antes de programar skills complejas a ciegas, cojo el PDF oficial de Anthropic y lo meto en NotebookLM. Lo uso para "entrenarme" y hacerle preguntas técnicas sobre la arquitectura antes de tocar una sola línea de código.

## 2. El Oráculo (Claude Code Best Practices)
Si usas Claude Code, este repo es obligatorio. Es mi manual de consulta de cabecera cada vez que tengo dudas, el agente se atasca o necesito optimizar mis prompts de sistema.

## 3. El Almacén de Piezas (Skills.sh, Antigravity y Awesome-Agent)
Cuando tengo una necesidad específica (un MCP o una skill concreta), no invento la rueda. Busco en estos tres directorios. Es como tener un LEGO gigante de funciones ya probadas para coger ideas.

## 4. La Base Teórica (Microsoft & AI For Beginners)
Y para esos días de investigación profunda, donde necesito entender el "porqué" de los modelos más allá del hype del momento, tiro de estas dos joyas.

Te dejo todos los enlaces ordenados aquí abajo para que no tengas que volverte loco buscándolos. 👇

¿Qué recurso o repositorio es tu "salvavidas" absoluto a la hora de trabajar con IA? ¡Recomiéndame uno y lo pruebo!
